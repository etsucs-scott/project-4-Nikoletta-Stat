# CSCI 1260 — Project 4

## Project Topic: Sudoku Game in Blazor Web App

Rules: Can't have the same cell value in any row, column, or 3x3 box.

How to play: 
1) Press the new game button for the timer to start.
2) Select an empty cell.
3) Click on the number you want to enter in the number keypad.
4) Fill the board!

When board is completed, the timer will stop and the DateTime of completion and elapsed seconds will be added to leaderboard.txt.

---

## Build and Run

In the terminal:
```bash
dotnet build
dotnet run --project src/Sudoku.UI
```

### Run Unit Tests
In Visual Studio --> Test Explorer --> Run all tests
